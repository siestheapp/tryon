## Age rating questionnaire — suggested answers (freestylefit)

This is a guide for **App Store Connect → Age Rating**. Final answers should reflect the shipped app.

### Suggested rating
**4+**

### Likely answers (typical for a try-on logging app)
- **Violence**: None
- **Profanity or crude humor**: None
- **Mature / suggestive themes**: None
- **Medical / treatment information**: None
- **Alcohol, tobacco, drugs**: None
- **Gambling**: None
- **Sexual content or nudity**: None
- **Horror/fear themes**: None
- **Frequent/intense contests**: None
- **Unrestricted web access**: No (the app accepts product URLs, but does not provide an in-app browser)
- **User-generated content**: Yes (users can enter notes). If asked whether content is shared publicly: **No** (private to the user).
